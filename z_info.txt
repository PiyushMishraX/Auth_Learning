DNS problem - It's required to write this dns import and setServer to 1.1.1.1 or 8.8.8.8 because  local Internet Service Provider (ISP) or local network router is failing to resolve the specialized DNS records required by MongoDB Atlas

fix for common js -
const dns = require("dns") // this is for common js
dns.setServers(["1.1.1.1","8.8.8.8"]);

fix for type module -
import dns from "dns"
dns.setServers(["1.1.1.1","8.8.8.8"])


bcrypt vs Crypto package
The primary difference is that bcrypt is a specialized library designed specifically for secure password hashing, while crypto (typically referring to Node.js’s built-in crypto module or the crypto-js library) is a general-purpose toolkit for various cryptographic operations like encryption, digital signatures, and generic hashing.
